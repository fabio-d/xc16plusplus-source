reloc057: BFD_RELOC_PIC30_FILE_REG (out-of-range):

The objective of this test is to verify that
out-of-range values for BFD_RELOC_PIC30_FILE_REG
relocations are detected properly.

How to execute the test:

1. Set the environment variable $PIC30_CD
    to your xc16 tools binary directory
2. Export $PIC30_CD
3. Execute "go"

To get verbose output:

Execute "go -v"
