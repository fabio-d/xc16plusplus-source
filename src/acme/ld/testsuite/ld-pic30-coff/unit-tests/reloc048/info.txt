reloc048: BFD_RELOC_PIC30_PWRSAV_MODE (out-of-range):

The objective of this test is to verify that
BFD_RELOC_PIC30_PWRSAV_MODE relocations detect
out-of-range values properly.

Note: This test shares source code with reloc046.

How to execute the test:

1. Set the environment variable $PIC30_CD
    to your xc16 tools binary directory
2. Export $PIC30_CD
3. Execute "go"

To get verbose output:

Execute "go -v"
