reloc077: BFD_RELOC_PIC30_TRAP_ARGUMENT (out-of-range):

The objective of this test is to verify that
BFD_RELOC_PIC30_TRAP_ARGUMENT relocations
detect out of range values properly.

NOTE: This test has been disabled by removing
the "go" script. To re-enable this test, copy
go.old to go and restart the test driver.

How to execute the test:

1. Set the environment variable $PIC30_CD
    to your pic30 tools binary directory
2. Export $PIC30_CD
3. Execute "go"

To get verbose output:

Execute "go -v"
