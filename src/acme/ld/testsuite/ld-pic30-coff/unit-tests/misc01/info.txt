misc01: Regression test of global initialized data:

The objective of this test is to verify that
global initialized data can be accessed by
the C compiler. The compiled program is
executed via the simulator and correct
output is verified.

How to execute the test:

1. Set the environment variable $PIC30_CD
    to your xc16 tools binary directory
2. Export $PIC30_CD
3. Edit the Configuration section of file "go"
4. Execute "go"

To get verbose output:

Execute "go -v"

File Descriptions:
-----------------
info.txt - this file
go - shell script to automate testing
t1.c - source file
run.cmd - command file for the simulator
test.out - output of the test
expect.out - correct output
