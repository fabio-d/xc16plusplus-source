cmd009: --no-data-init option:

The objective of this test is to verify that
a data init template is not created when the
--no-data-init option is specified. Note that
a warning should be issued by the linker if
initialized data is present.

How to execute the test:

1. Set the environment variable $ACME_BIN
    to your xc16 tools binary directory
2. Export $ACME_BIN
3. Edit the Configuration section of file "go"
4. Execute "go"

To get verbose output:

Execute "go -v"

File Descriptions:
-----------------
info.txt - this file
